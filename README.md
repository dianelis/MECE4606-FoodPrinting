# MECE 4606 — Food Printing Assignment

**Course:** MECE 4606 — Digital Manufacturing  
**Semester:** Spring 2026  
**University:** Columbia University

---

## 📋 Overview

This project explores the low-level mechanics of digital manufacturing through food-based 3D printing. Using paste extruders and custom G-code, we fabricate increasingly complex edible structures — from simple squares to multi-material spirograph patterns.

---

## 🎯 Assignment Parts

### Part 1 — Square Print
Prepare a G-code file for printing a **1-inch square** and print it using a food material (e.g., cream cheese).  
> *Source code not required for this section.*

### Part 2 — Twisted Triangle (Pyramid)
Fabricate a **twisted triangle** 3D structure by printing stacked triangles, rotating and shrinking each layer. Optionally print two tightly-nested triangles per layer for added rigidity. Use a **different material** than Part 1.

### Part 3 — Multi-Material Structure
Repeat the twisted pyramid using **two materials** (e.g., switching material every layer).

### Part 4 — Spirograph / Intricate Shape
Fabricate an intricate shape by moving the print-head along an **epicycloid trajectory** (Spirograph pattern). Choose parameters to generate an aesthetic 3D shape. Use **multiple ingredients** by switching out syringes.

## 🏭 Logistics

- **Lab Location:** MechTech Lab (ET 273) or MechE Lab
- **Equipment:** Paste extruders with reusable dispensers and nozzles (provided by TA)
- **Budget:** $50 per team for food materials (keep all receipts)
- **Storage:** Label bins in the refrigerator with full name, UNI, and phone number
- **Machines:** First-come, first-served basis

### ⚠️ Important Rules
- **Clean dispensers and nozzles** thoroughly after each use
- **Wipe the printer** clean of all food material before and after use
- **Do NOT eat food off the printer** — the machine is not food safe
